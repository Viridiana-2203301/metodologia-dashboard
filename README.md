# 📊 Plataforma de Visualización - Defectos Congénitos en México

Plataforma web interactiva para la visualización y análisis de datos sobre defectos congénitos en México, implementada con **Flask** y siguiendo metodología **Data Vault**.

## 🎯 Descripción del Proyecto

Este proyecto implementa un sistema de visualización de datos que integra información de:
- **Defunciones**: Casos de defectos congénitos que resultaron en defunciones
- **Nacimientos**: Datos de nacimientos y características del parto




## 🚀 Instalación y Uso


### Paso 1: Activar el Entorno Virtual

**En Windows (CMD):**
```bash
.\.venv\Scripts\activate.bat
```

### Paso 2: Instalar Dependencias

```bash
pip install -r requirements.txt
```
